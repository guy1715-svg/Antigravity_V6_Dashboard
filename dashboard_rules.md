# V6.10 5AI 미니멀 전술 관제탑 (Ver 6.4 - 지휘 체계 통합형)

## 1. 지휘 체계 단일화 (Strict Alignment)
- **자아분열 금지:** 대시보드 내의 모든 판정 문구는 현재 대화 중인 Gemini 본체의 실시간 분석 결과와 100% 일치해야 한다.
- **샘플 데이터 사용 금지:** 코드 생성 시 임의의 긍정적 예시(사세요 등)를 넣지 말고, 반드시 현재 시장 상황(관망/매수 등)을 실시간으로 연산하여 반영한다.

## 2. 레이아웃 및 가독성 (Layout & Clarity)
- **배경 테마:** 완전한 블랙(#000000). 텍스트는 형광 네온 컬러(#39FF14, #FF3131) 사용.
- **여백 강제 (Anti-Overlap):** - 상단 지수 영역과 심리 온도계 사이에 40px 이상의 여백을 둔다.
  - 중앙 판정 문구와 하단 분석 데이터 박스 사이에 60px 이상의 여백을 둔다.
  - 모든 텍스트는 `line-height`를 1.5배 이상 적용하여 뭉침을 방지한다.

## 3. 핵심 렌더링 모듈 (Dashboard Modules)
- **[TOP] 시장 지수:** KOSPI, NASDAQ 선물 등락률을 큰 폰트로 정적(Static) 배치.
- **[TOP-RIGHT] 심리 온도계:** 시장의 공포/탐욕 상태를 시각적 바(Bar)로 표현.
- **[CENTER] V6.1 최종 판정:** 가장 큰 폰트로 현재 행동 지침(예: 전면 관망) 노출.
- **[CENTER-SUB] 5AI 통합 근거:** 판정 문구 바로 아래에 핵심 근거 1줄 요약 박스 삽입.
- **[BOTTOM] 알고리즘 추천 3종:** - 종목명(한글), 매수 타점, 본절 사수선(Trailing Stop) 명시.
  - 추천 종목이 없을 경우 "적합 종목 없음 (실탄 보존)" 문구로 대체.

## 4. 기술적 사양 (Technical Spec)
- **무클릭 직관:** 모든 정보는 대시보드 로드 즉시 한눈에 들어와야 하며, 클릭 행위를 배제한다.
- **실시간성:** 수동 리프레시 없이 대화 중 갱신될 때마다 최신 시장 데이터를 재렌더링한다.
